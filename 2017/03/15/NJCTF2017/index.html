这次出了很多Web题目都很有意思，很多都没做出来，赛后总结一下吧

Web

Login

打开发现让登录，右下可以注册，直接想到去年ISCC的mysql超长字符串截取，注册admin q,由于mysql只截取前十六位所以直接用admin就能登陆获取flag。
